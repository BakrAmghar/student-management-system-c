# 🎓 Student Management System v2.0
A high-performance student database application built with a **C Linked-List Backend** and a **Qt 6 Graphical Frontend**.

## 🚀 The Evolution of the Project
This project started as a challenge to master low-level memory management and has evolved into a feature-rich desktop application:

1.  **Phase 1: Pure C Foundation** - Started with basic C logic using `struct` and pointers.
    - Implemented a **Doubly Linked List** for efficient student data handling.
    - Built custom File I/O for data persistence (`.txt`).
    - *Result:* A solid, fast command-line database.

2.  **Phase 2: GUI Migration (FLTK & Qt)**
    - Transitioned from a terminal interface to a modern GUI.
    - Explored FLTK before moving to **Qt 6** for its professional-grade tools and UI designer.
    - Integrated C backend logic with C++ frontend signals and slots.

3.  **Phase 3: Data Visualization & Export**
    - Added **QtCharts** for real-time grade distribution analytics.
    - Implemented **Live Search** and filtering.
    - Integrated a **Report Generator** for PDF and Excel/CSV output.

## 🛠️ Tech Stack
* **Language:** C (Core Logic) & C++ (Interface)
* **Framework:** Qt 6.10 (Widgets, Charts, PrintSupport)
* **Build System:** CMake
* **Data Structure:** Doubly Linked List

## ✨ Current Features
* ✅ **ID Security:** Prevents duplicate student entries at the logic level.
* 📊 **Live Analytics:** Dynamic bar charts update as you add or filter students.
* 🔍 **Instant Search:** Real-time filtering of the student list by name or ID.
* ⚙️ **Advanced Management:** Contextual popups to modify or delete records.
* 📤 **Professional Export:** Generate official PDF reports or Excel-compatible CSV files.
