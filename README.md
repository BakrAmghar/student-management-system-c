# Student Management System (C)

This is a **Student Management System** built as a **console application in C**.
It allows managing student records using **linked lists** and provides basic functionality such as adding, deleting, searching, and calculating averages.

---

## Current Features

* Add new students
* Delete student records
* Search for students
* Display all students
* Calculate student averages

---

## Technologies Used

* **C Programming Language**
* **Linked Lists**
* **Code::Blocks**
* **Git & GitHub**

---

## Project Goal

The goal of this project is to practice:

* **Data structures in C**
* **Modular programming**
* **File organization in C projects**
* **Version control with Git**
